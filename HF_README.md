# CRREM Portfolio Assessment — Preview

Methodology demonstration built for Duff & Phelps Investment Management.

Five listed REITs (LAND.L, URW.PA, PLD, VNA.DE, SGRO.L) modelled as
CRREM-compliant mini-portfolios of pseudo-assets, assessed against the
1.5°C decarbonisation pathway through 2050, classified through the IIGCC
Net Zero Investment Framework, presented through four audience lenses
(Investment Committee, Regulatory/LP, Engagement, Analyst).

Inputs are illustrative placeholders — methodology is real.
